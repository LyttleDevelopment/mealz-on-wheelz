# Mealz on Wheelz

A root-level Next.js landing page for Mealz on Wheelz, built mobile-first and wired to the local design framework in `packages/design-framework/packages/ui`.

## Stack

- Next.js App Router
- React 19
- Sass modules
- Local `@lyttle-development/ui` dependency

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run lint
```

## Notes

- Shared brand tokens were updated in `packages/design-framework/packages/ui/src/styles/_themes.scss`.
- The website entrypoint lives in `app/page.tsx`.
